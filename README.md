# Data Cleaning 

This project provides a complete data cleaning pipeline for preparing datasets.

## Features
- Data inspection
- Missing value handling
- Duplicate removal
- Outlier detection (IQR method)
- Data standardization (text, numbers, dates)
- Clean dataset output

## Tech Stack
- Python
- Pandas
- NumPy

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run cleaning script
python src/clean_data.py

## Output
Cleaned dataset will be saved in:
output/cleaned_data.csv
